0. 가상환경 생성

python -m venv mysite


1. 장고 설치

pip install django


2. django-admin을 이용하여 장고 프로젝트 생성하기

django-admin startproject config .


3. 웹 사이트 구동하기

python manage.py runserver


4. django-admin의 startapp 명령을 이용하여 pybo앱을 생성하기

django-admin startapp pybo


5. 프로젝트 생성시 기본적으로 설치되는 앱들이 필요로 하는 테이블(데이터베이스에서 저장하기 위한 데이터 집합의 모임) 생성하기

python manage.py migrate


6. 모델에서 테이블 생성 후 makemigrations 명령 수행하기

python manage.py makemigrations


7. makemigrations 실행 후 실제 테이블 생성하기

python manage.py migrate
